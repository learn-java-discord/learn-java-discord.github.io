About this FAQ

This FAQ was created by the Learn Java Discord Channel to help anyone new with Java and those looking for guidance. There are many challenges with learning the Java ecosystem and the JVM stack, but hopefully this guide gives a coherent introduction to the language.

Where do I start?

This depends on your goals as a Java programmer. The sections below gives some advice depending on whether your priority is mobile development, web development, or enterprise development. Enterprise development is particularly important if you are simply looking for work. Regardless, starting with vanilla Java is a given.
